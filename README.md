# Baykul Frontend

This is the frontend application for the Baykul Auto Parts Marketplace.

## Documentation

Please refer to [PLAN.md](./PLAN.md) for the detailed implementation plan, architecture, and API specifications.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build**:
    ```bash
    npm run build
    ```
